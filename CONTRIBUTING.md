## 📋 BEFORE YOU CONTRIBUTE

### Step 1: Read the Vision
- [README.md](README.md) - What this project IS
- [ROADMAP.md](ROADMAP.md) - Where we're going

### Step 2: Check Existing Issues
- Look through [open issues](https://github.com/quantum-weaver/audhdities/issues)
- Someone may already be discussing your idea
- Duplicates are okay—just adds weight to the idea

### Step 3: Use the Right Template
When opening an issue, please use one of these prefixes:

| Prefix | For |
|:---|:---|
| `[BUG]` | Something isn't working |
| `[IDEA]` | A new feature or enhancement |
| `[DESIGN]` | UI/UX feedback or suggestions |
| `[QUESTION]` | You need clarification |
| `[DOCS]` | Documentation improvements |

## 📝 STYLE GUIDELINES

When you do contribute, please keep these in mind:

### For Issues:
- Clear, descriptive title
- Steps to reproduce (for bugs)
- What you expected vs. what happened
- Screenshots if helpful
- Your neurodivergent context is welcome (e.g., "This UI element is hard for me because...")

### For Code (when invited):
- Follow existing patterns in the codebase
- Comment generously—future neurodivergent coders will thank you
- Tests are appreciated but not required initially
- TypeScript is required (no `any` unless absolutely necessary)

### For Documentation:
- Simple words. Short sentences.
- Assume the reader is brilliant but overwhelmed.
- Examples > explanations.
- Humor is allowed. Even encouraged.

## 🔄 THE CONTRIBUTION LIFECYCLE

```
1. IDEA ──► 2. DISCUSSION ──► 3. ISSUE ──► 4. IMPLEMENTATION ──► 5. REVIEW ──► 6. MERGE
```

| Stage | What Happens |
|:---|:---|
| **Idea** | You have a thought. Write it down before it evaporates. |
| **Discussion** | Open a Discussion or issue with `[IDEA]`. We talk. We refine. |
| **Issue** | If the idea is solid, it becomes a tracked issue with clear acceptance criteria. |
| **Implementation** | Code happens. Or docs. Or design. |
| **Review** | The Quantum Weaver (and eventually others) reviews. Feedback is kind. |
| **Merge** | Your contribution becomes part of the sanctuary. Forever. |
